# AIPromptHistoryTimeline

A clean, minimal UI timeline for displaying past prompts and responses. Useful for any AI app (chat, summarizer, generator) to help users track previous interactions with an AI model. Pure frontend, stores prompts locally in state or localStorage. No backend required.

## Features
- Shows previous prompts and model outputs (with timestamps)
- Scrollable, visually engaging timeline
- Copy buttons to reuse prompt or response
- Option to clear history
- Local-only (no DB)

## Props

| Prop      | Type                                                                 | Description                                 |
|-----------|----------------------------------------------------------------------|---------------------------------------------|
| items     | `{ prompt: string; response: string; timestamp?: number }[]`         | Array of prompt/response items              |
| onClear   | `() => void` _(optional)_                                            | Function to clear history                   |
| className | `string` _(optional)_                                                | Additional CSS classes                      |
