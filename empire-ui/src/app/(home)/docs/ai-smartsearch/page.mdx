# AI Chatbox

---

A flexible AI-powered chatbox component that can handle user inputs and display responses from an AI. This chatbox is interactive and supports features like smooth scrolling, avatars, and a user-friendly interface.

---

/>

## Props

---

| Prop name   | Type      | Description                                     |
| ----------- | --------- | ----------------------------------------------- |
| `className` | `string`  | The class name for the chatbox container.       |
| `isOpen`    | `boolean` | Controls whether the chatbox is open or closed. |
