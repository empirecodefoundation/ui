# Voice Command Button

---

A mic-enabled button that listens to voice commands and triggers pre-defined UI actions â€” powered entirely by the browser-native Web Speech API. No external API key required.

> **Note:** The Voice Command Button uses the [Web Speech API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Speech_API), which is supported in most modern browsers (Chrome, Edge). It does not require any external API key.

    "scroll down": () => window.scrollBy(0, 500),
    "scroll up": () => window.scrollBy(0, -500),
    "dark mode": () => document.body.classList.add("dark"),
    "light mode": () => document.body.classList.remove("dark"),
  }}
  onCommandRecognized={(cmd) => console.log("Recognized:", cmd)}
/>
`} lang="tsx" />

## Props

---

| Prop name             | Type                                 | Description                                              |
|----------------------|--------------------------------------|----------------------------------------------------------|
| `commands`           | `Record<string, () => void>`          | Mapping of keywords/phrases to callback functions         |
| `onCommandRecognized`| `(command: string) => void`           | Optional callback when a command is matched               |
| `onError`            | `(error: string) => void`             | Optional callback on error                                |
| `label`              | `string`                              | Optional label under the button                           |
