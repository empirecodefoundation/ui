# Khul Ja Sim Sim Door

---

An animated, interactive gate-like component inspired by the concept of magical doors (like in Aladdin). The AI Door challenges users with personalized questions and only opens when the correct answer is provided. Wrong answers generate a new challenge. Useful for creative authentication, onboarding, or gamified access flows.

## Props

---

| Prop name         | Type     | Description                                         |
| ----------------- | -------- | --------------------------------------------------- |
| `userName`        | `string` | The user's name shown after successful gate access. |
| `initialQuestion` | `string` | The first question shown to the user.               |
