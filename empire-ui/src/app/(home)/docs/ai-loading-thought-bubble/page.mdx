# Loading Thought Bubble

---

A fun, animated "thinking" bubble for AI processing or data loading. Cycles through quirky, AI-themed messages with animated dots to entertain users while they wait.

> **Note:** This component is perfect for showing AI/LLM loading states with personality. No external API required.

## Props

---

| Prop name   | Type        | Description                                              |
|-------------|-------------|----------------------------------------------------------|
| `messages`  | `string[]`  | Array of messages to cycle through (optional, AI defaults) |
| `interval`  | `number`    | Time in ms between message changes (default: 3000)      |
| `className` | `string`    | Optional Tailwind classes for custom styling            |
